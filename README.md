# kotlin-plugin
Allows to use kotlin in your plugins.
### Libraries
  - kotlin
  - kotlin-reflect
  - kotlinx-coroutines
### Installation
  - git clone https://github.com/OtekPlay/kotlin-plugin
  - mvn clean install
  - copy file KotlinPlugin.jar to your /plugins
### How to use
Just add to plugin/bungee yaml file.
`depend: [KotlinPlugin]`
